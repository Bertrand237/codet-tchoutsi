Tu es un développeur full-stack Firebase expert (Firebase Auth, Firestore, Storage, Hosting, Functions, et Realtime Database). Ta mission est de créer une application web complète, moderne et entièrement hébergée sur Firebase, appelée CODET (Comité de Développement Tchoutsi). Le résultat attendu est un projet complet prêt à déployer avec Firebase Hosting, utilisant uniquement les services Firebase pour la base de données, le stockage et l’authentification. 🎯 Objectif du projet Concevoir une application web verte, intuitive et responsive, permettant : La gestion des membres et de leurs cotisations ; La validation des paiements avec preuve (PDF/Image) ; Le recensement familial ; Un réseau social interne type WhatsApp (messages en temps réel, groupe unique CODET) ; Un blog public administrable ; Un espace publicitaire vidéo jouant en boucle sur la page d’accueil ; Un tableau de bord analytique visible par le bureau exécutif. 🧱 Technologies à utiliser Frontend : HTML5, CSS3, JavaScript (ou Vue.js/React si nécessaire) Base de données : Firebase Firestore Stockage : Firebase Storage (preuves de paiement + publicités vidéo) Authentification : Firebase Auth (Email + mot de passe) Cloud Functions : Pour validation, notifications et statistiques automatisées Hébergement : Firebase Hosting Notifications : Firebase Cloud Messaging 🌿 Thème & UX Thème principal vert (#0A7D33) et blanc ; Design clair, moderne, avec sidebar, icônes FontAwesome et cartes ; Interface responsive mobile/desktop. 🧩 Modules à développer 1️⃣ Authentification & gestion utilisateurs Connexion / inscription avec Firebase Auth (Email + Mot de passe). Rôles enregistrés dans Firestore : admin, président, trésorier, commissaire, membre, visiteur. Redirection automatique selon rôle. 2️⃣ Paiements & validation Collection Firestore : paiements Champs : membreId, montant, date, mode, preuveURL, statut. Upload de preuve (image/PDF) → Firebase Storage. Validation par commissaire (mise à jour du champ statut : validé / rejeté). 3️⃣ Recensement familial Collection Firestore : recensements Champs : membreId, nom_enfant, date_naissance, sexe, lieu_naissance, etat_civil, niveau_scolaire, profession. Interface CRUD accessible au membre et consultable par le bureau exécutif. 4️⃣ Réseau social interne (type WhatsApp) Collection Firestore : messages Tous les utilisateurs inscrits sont automatiquement ajoutés au groupe unique “CODET Général”. Impossible de quitter le groupe. Interface : Liste des messages (pseudo, texte, heure). Envoi de texte, image, audio ou PDF. Affichage en temps réel via Firestore onSnapshot(). Notifications via Firebase Cloud Messaging pour chaque nouveau message. 5️⃣ Blog public Collection : articles Champs : titre, contenu, imageURL, auteur, datePublication. Consultation libre. Gestion des articles dans le back-office admin. 6️⃣ Espace publicitaire vidéo Collection : publicites Champs : titre, videoURL, ordre. Affichage : fenêtre moyenne sur la page d’accueil. Les vidéos tournent en boucle automatique (HTML5 <video autoplay loop muted>). Upload et gestion via Storage + interface admin. 7️⃣ Tableau de bord & statistiques Graphiques via Chart.js : Total membres Total paiements validés Montant total cotisé Nombre d’enfants recensés Messages échangés Données récupérées en temps réel depuis Firestore. 🔐 Rôles et permissions Admin : accès complet Président : gestion tâches, blog, supervision Trésorier : paiements Commissaire : validation Membre : paiements, recensement, messagerie Visiteur : lecture blog 🧰 Structure du projet Firebase code Code CODET-Firebase-App/ ├── index.html ├── dashboard.html ├── chat.html ├── recensement.html ├── blog.html ├── style.css ├── script.js ├── firebase-config.js ├── /images/ ├── /videos/ └── README.md 📂 Fichiers clés à inclure firebase-config.js → Contient les variables Firebase SDK. firestore.js → Logique de lecture/écriture des collections. auth.js → Inscription / connexion / rôles. chat.js → Messagerie temps réel (Firestore + FCM). ads.js → Gestion des publicités vidéo. recensement.js → Formulaire et gestion du module familial. ⚙️ Livrables demandés Fournis : Tout le code HTML/CSS/JS complet Firebase configuration prête à coller dans un vrai projet README.md expliquant : Comment connecter le projet à Firebase.
